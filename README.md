# Business-Case-Analysis-Using-SQL
Overview:
    This project focuses on solving real-world business problems using SQL on a large-scale E-commerce dataset. The goal was to analyze     revenue, customer behavior, and product performance to generate actionable insights.

Tools Used:
    MySQL

Dataset:
Online Retail Dataset
500,000+ records
Real-world transactional data

Data Cleaning
  Performed using SQL:
    Removed NULL customer IDs
    Filtered negative quantities (returns)
    Removed zero-price transactions
    Cleaned and structured raw transactional data
    
Business Questions Solved:
  1. Revenue Analysis
  Calculated total and monthly revenue
  💡 Insight: Revenue shows clear seasonal patterns
  2. Product Performance
  Identified top and low-performing products
  💡 Insight: A small number of products generate most revenue
  3. Customer Analysis
  Identified top customers and repeat buyers
  💡 Insight: High-value customers drive a large portion of sales
  4. Country-wise Revenue
  Compared revenue across countries
  💡 Insight: Revenue is concentrated in a few regions
  5. Customer Behavior
  Analyzed repeat vs one-time customers
  💡 Insight: Customer retention is a key growth factor
  6. Average Order Value
  Calculated average spending per order
  💡 Insight: Helps evaluate purchasing patterns
